# 🎨 План создания Placeholder-арта (ART-S1-001)

Так как Unity Editor в данный момент закрыт (MCP отключен), я:

1. **Создаю UI-иконки** (Огнемёт и Крио) с помощью встроенной генерации изображений. Они будут переданы UIUX для кнопок.
2. **Создаю Editor-утилиту в Unity**. Как только вы откроете Unity, нужно будет нажать `GameDev Squad -> Generate ART Placeholders` в верхнем меню.
3. Утилита автоматически создаст папку `Assets/Art/Placeholders`, сгенерирует 10+ цветных материалов согласно ТЗ и соберёт из примитивов (Кубы, Цилиндры) готовые префабы (Башни Lv1-3, 2 Крипа, сетку).

## Требования стиля

- **Огнемёт (Incinerator):** Оранжево-красный. Приземистый силуэт. Куб + Горизонтальный цилиндр.
- **Крио (Cryo):** Сине-голубой. Узкий высокий силуэт. Вертикальный цилиндр + Сфера сверху.
- **Крип 1 (Fast):** Маленький обтекаемый (Капсула, зелёный).
- **Крип 2 (Tank):** Большой широкий (Сплюснутый куб, фиолетовый).

Я приступаю к написанию `PlaceholderArtGenerator.cs` (Editor Script).
