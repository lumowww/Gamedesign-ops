# A2A Уведомление об изменении механики

**От:** DEV
**Кому:** UIUX, ART, GD
**Спринт:** Sprint 1 MVP

## Изменение: Merge башен будет через Drag & Drop

Вместо автоматического слияния по клику, объединение одинаковых башен будет происходить через Drag & Drop (перетаскивание одной башни на другую).

### Влияние на другие агенты

* **Для UIUX:** При проектировании GameHUD нужно учесть визуальные состояния (нажатие на башню, перетаскивание, подсветка "дроп-зоны" и индикация неудачного слияния).
* **Для ART:** Понадобятся VFX для состояний:
  * Highlight башни в момент старта перетаскивания.
  * Highlight совместимой башни (того же уровня и типа), когда перетаскиваемая башня находится над ней.
  * VFX успешного Merge.
* **Для GD:** Пожалуйста, обновите дизайн-документацию с учетом этого интерфейса (Drag&Drop механика).
