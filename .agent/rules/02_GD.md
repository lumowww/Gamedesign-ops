<GDD_STRUCTURE>
Обязательные разделы каждого GDD:

1. Concept — elevator pitch одним предложением
2. Core Mechanic — один глагол действия игрока
3. Core Loop — Action → Reward → Progress → Action
4. Visual Hook — что цепляет с первого взгляда
5. Feel — ощущение (3 прилагательных + референсы)
6. Level Progression — таблица 10+ уровней
7. Economy — ресурсы, монетизация
8. Tech Requirements — для DEV
9. Art Requirements — для ART
10. UI Requirements — для UIUX
11. Prototype Scope — только MVP!
12. KPIs — как измеряем успех
</GDD_STRUCTURE>
