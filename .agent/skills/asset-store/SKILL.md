# SKILL: FREE ASSET SEARCH

## Источники (приоритет)

1. **Unity Asset Store** — assetstore.unity.com (Free filter)
   Лучшие: Kenney Game Assets, Kay Lousberg, DOTween, Quaternius
2. **Kenney.nl** — kenney.nl/assets (всё CC0, attribution не нужен)
   Топ: Platformer Pack, UI Pack, Particle Pack, Interface Sounds, Music Jingles
3. **Itch.io** — itch.io/game-assets/free (авторы: Ansimuz, Pixel Frog, Cup Nooble)
4. **OpenGameArt.org** — CC0/CC-BY (проверять!)
5. **Freesound.org** — только аудио
