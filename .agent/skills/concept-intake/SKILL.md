# SKILL: CONCEPT INTAKE & EVALUATION

## Что делает этот скилл
Продюсер оценивает концепт-документ по трём осям:
1. **Полнота информации** — есть ли всё необходимое для старта
2. **Marketability Score** — потенциал CPI/Retention
3. **Feasibility Score** — реализуемость соло AI-командой

## ПРАВИЛА ОЦЕНКИ MARKETABILITY

### Visual Hook (CPI-потенциал)
- 9-10: Визуально понятно за 1 секунду, есть WTF-момент
- 7-8: Чётко читается за 2 секунды
- 5-6: Нужно объяснение — риск высокого CPI
- < 5: Переосмыслить визуал

### Mechanic Simplicity
- 9-10: Один глагол, нулевое обучение (Flappy Bird уровень)
- 7-8: Туториал за 10 секунд
- 5-6: Туториал 30+ секунд — риск для HC
- < 5: Слишком сложно для HC, возможно OK для Hybrid

### Trend Alignment (проверяем по Top Charts)
- 9-10: В топ трендах прямо сейчас + есть свой twist
- 7-8: Стабильный жанр, не перенасыщен
- 5-6: Жанр угасает или очень конкурентный
- < 5: Против тренда или сверхконкурентная ниша

## РЕФЕРЕНСНЫЙ АНАЛИЗ (алгоритм)

При наличии референсов (ссылки, названия игр):

1. **Реверс-инжиниринг Core Loop:**
   - Что делает игрок? (глагол)
   - Что получает немедленно? (feedback)
   - Что копится? (progress)
   - Что возвращает завтра? (retention hook)

2. **Идентификация дифференциатора:**
   - В чём ваша версия лучше/иначе?
   - Что можно улучшить в механике?
   - Какой арт-стиль победит референс по CPI?

3. **Конкурентный анализ:**
   - Сколько похожих игр в топе? (если >5 — высокая конкуренция)
   - Когда вышел последний хит в жанре? (тренд жив?)
   - Есть ли "белое пятно" для дифференциации?
