# WORKFLOW: /handoff

# Версия: 2.0 | Инициатор: Текущая модель LLM

## Контекст

Выполняется **перед** тем, как пользователь переключит активную LLM-модель в интерфейсе IDE (например, с Gemini на Claude Sonnet), либо перед длительной паузой в работе.

## Задача Агента

1. Создать (или перезаписать) файл `.gemini/antigravity/brain/session_handoff.md`.
2. Сформировать "слепок" текущего состояния по шаблону ниже.

## Шаблон session_handoff.md

```markdown
# SESSION HANDOFF
**Timestamp:** [Текущая дата и время]
**Последняя активная модель:** [Твое имя/версия]
**Текущий Workspace/Роль:** [Например, DEV-Workspace]

## Чем мы занимались (Активная задача)
[Коротко (2-3 предложения) опиши текущую техническую или дизайн-цель]

## На чём остановились (Конкретный шаг)
- [Что было только что написано/изменено]
- [Где находится курсор фокуса (название файла/класса)]

## Известные проблемы / Ошибки (Если есть)
- [Лог последней ошибки из консоли Unity или терминала]
- [Неработающий кусок кода]

## Что должна сделать СЛЕДУЮЩАЯ модель
1. [Шаг 1, который ты планировал сделать]
2. [Шаг 2...]
```

## Завершение

После записи файла, выведи в чат сообщение:
> 💾 **Контекст сохранен в `session_handoff.md`.** Можете безопасно переключать модель. После смены модели напишите `/resume`.
