# Триггер: /new-project

# Агент-инициатор: PM

## ФАЗА 1 — Сбор информации (PM)

Запросить у пользователя:

- Название проекта
- Концепт (одно предложение или ссылка на референс)
- Целевая платформа (iOS / Android / Both)
- Дедлайн прототипа
- Ограничения (бюджет на ассеты, жанровые рамки)

## ФАЗА 2 — Инициализация PM

Создать `.gemini/antigravity/brain/project_context.md`:

```
# PROJECT: [Name]
Дата старта: [дата]
Дедлайн прототипа: [дата]
Концепт: [одно предложение]
Платформа: iOS / Android
Текущий Sprint: Sprint 1
Статус: 🟡 ИНИЦИАЛИЗАЦИЯ
```

Создать начальный `task_board.md` (Sprint 1).

## ФАЗА 3 — Задачи агентам

- TASK-001 → GD: GDD v0.1 (Concept + Core Mechanic + Prototype Scope), ETA День 1
- TASK-002 → ART: Арт-библия v0.1, ETA День 1-2
- TASK-003 → UIUX: Список экранов + UI структура Figma, ETA День 2
- TASK-004 → DEV: Настройка Unity проекта (структура + пакеты + MCP), ETA День 1

## ФАЗА 4 — Первый /sync

После выполнения TASK-001..004 провести /sync, выдать Sprint 1 задачи.
