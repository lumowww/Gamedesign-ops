# WORKFLOW: Qwen Autonomous Mode

**Команда:** `/qwen-autonomous [TASK-ID]`

**Описание:** Автономное выполнение задач Qwen Code в рамках мультиагентной системы Antigravity

## ПРЕ-УСЛОВИЯ
- [ ] Задача определена в `.qwen/tasks/`
- [ ] Конфигурация `.qwen/QWEN.md` прочитана
- [ ] Роль агента определена (PM/DEV/ART)

## АЛГОРИТМ

### 1. Определение роли
```
1. Проверить `.agent/rules/` — активный rule-файл
2. Проверить `.gemini/antigravity/brain/project_context.md`
3. Проверить переменную окружения `AGENT_ROLE`
4. Анализовать структуру проекта
```

**Результат:**
- `PM` → чтение, документация, задачи
- `DEV` → код, Git, сборка, локальные сервисы
- `ART` → ассеты, UI

### 2. Проверка контрактов (ABC)
```
Предусловия → Инварианты → Констрейнты → Восстановление
```

### 3. Оценка риска
| Уровень | Действие | Требуется подтверждение |
|---------|----------|-------------------------|
| НИЗКИЙ | venv, npm install, docker restart | ❌ |
| СРЕДНИЙ | cache clean, docker-compose restart | ❌ + лог |
| ВЫСОКИЙ | .lock файлы, конфиги | ✅ |
| ОЧЕНЬ ВЫСОКИЙ | prod конфиги, git reset | 🚫 ЗАПРЕЩЕНО |

### 4. Выполнение
```powershell
# PowerShell синтаксис (Windows)
Test-Path <path>
Get-ChildItem <dir>
New-Item -Path <file> -ItemType File -Force

# Логирование каждого шага
Add-Content -Path ".gemini/antigravity/brain/decisions_log.md" -Value "[timestamp] — [action]"
```

### 5. Пост-условия
- [ ] Задача отмечена выполненной в `.qwen/tasks/`
- [ ] Лог записан в `decisions_log.md`
- [ ] Git commit сделан (если изменялись файлы)
- [ ] Следующий агент уведомлён (если handoff)

## ВОССТАНОВЛЕНИЕ ПРИ ОШИБКАХ

1. **PowerShell ошибка:** Проверить execution policy
2. **Git ошибка:** Проверить credentials, remote
3. **Docker ошибка:** Проверить daemon, контейнеры
4. **Неизвестная ошибка:** Создать тикет, приложить логи
