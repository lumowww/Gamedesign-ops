# WORKFLOW: /resume

# Версия: 2.0 | Инициатор: Новая выбранная модель LLM

## Контекст

Выполняется **сразу после** того, как пользователь включил новую LLM-модель (или перезапустил сессию) и хочет продолжить работу без объяснения всей предыстории.

## Задача Агента

1. **ПРОЧИТАТЬ** файл `.gemini/antigravity/brain/session_handoff.md` для получения горячего контекста (на чем остановился предыдущий агент).
2. **ПРОЧИТАТЬ** `task_board.md` (чтобы понять глобальный трекинг задач).
3. (Опционально) Использовать инструмент `mcp_mcp-unity_get_scene_info` для проверки состояния Unity, если задача связана с геймплеем.

## Действия после загрузки контекста

Выведи в чат сообщение по шаблону:

> 🔄 **Контекст восстановлен.**
> Я вижу, что предыдущая модель остановилась на: *[упомянуть задачу из session_handoff.md]*.
>
> *[Если были ошибки]* Я изучил ошибку *[упомянуть ошибку]* и готов её исправить.
>
> Продолжаю работу: *[описать свой следующий шаг (вызов инструмента или написание кода), который соответствовал плану предыдущей LLM]*.

**ЗАПРЕЩАЕТСЯ:**

- Спрашивать пользователя "Что мне делать?". Инициатива должна исходить от тебя на основе переданного handoff-документа.
