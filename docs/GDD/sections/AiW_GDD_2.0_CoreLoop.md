## 📊 Overview

Core Loop — это цикл действий, который игрок повторяет на протяжении всей игры. В AiW мы используем **трёхуровневую модель**:

```
┌────────────────────────────────────────────────────────┐
│  MACRO: Прогрессия аккаунта / Сезонный контент        │
│    ↑                                                   │
│  META: Прокачка колоды / Герои / PvP рейтинг          │
│    ↑                                                   │
│  MICRO: Матч (спавн → атака → merge → победа)         │
└────────────────────────────────────────────────────────┘
```

---

## 🎮 MICRO Loop (В матче)

**Длительность:** 3-5 минут

**Цикл:**
1. **Спаун башни** → Игрок тратит очки на спаун башни на grid
2. **Атака** → Башня автоматически атакует крипов в радиусе
3. **Уничтожение** → Крип умирает → игрок получает очки
4. **Merge** → Игрок сливает 2 одинаковые башни → новая башня Lv+1
5. **Повтор** → Новые волны крипов → больше очков → больше башен

**Win Condition:** Пройти 10 волн

**Lose Condition:** 10 крипов достигли финиша

**Ключевые механики:**
- SpawnManager: спаун башен по клику
- MergeGrid: логика слияния (2→1, Lv N→Lv N+1)
- PathFollower: движение крипов по П-образной тропе
- ScoreSystem: начисление/списание очков

---

## 📈 META Loop (Между матчачами)

**Длительность:** 10-30 минут

**Цикл:**
1. **Матч** → Игрок завершает матч (победа/поражение)
2. **Награда** → Получает шарды башен + софт-валюту
3. **Апгрейд** → Тратит шарды на апгрейд башен в колоде
4. **Прогресс** → Открывает новые башни/героев
5. **Следующий матч** → Сложнее, больше наград

**Ключевые механики:**
- Tower Upgrade: шарды + софт-валюта → уровень башни
- Deck Building: выбор 4 башен для матча
- Hero System: герой с уникальной способностью

---

## 🏆 MACRO Loop (Прогрессия аккаунта)

**Длительность:** Недели/месяцы

**Цикл:**
1. **Игра** → Игрок играет матчи, получает награды
2. **PvP Рейтинг** → Поднимается в лигах (Bronze → Silver → Gold)
3. **Battle Pass** → Выполняет квесты → уровни BP → награды
4. **Сезон** → Конец сезона → сброс рейтинга → новые награды
5. **Retention** → Daily Quests → возвращение в игру

**Ключевые механики:**
- PvP Matchmaking: по tower tier (сумма уровней башен)
- Battle Pass: 50 уровней, бесплатные + premium награды
- Daily Quests: 3 квеста в день, сброс через 24ч
- Season Reset: сброс рейтинга, новые сезонные награды

## 🎯 Design Heuristics (Проверка)

| Правило | Статус | Примечание |
|---------|--------|------------|
| **Ясная цель** | ✅ | Пройти 10 волн |
| **Обратная связь** | ✅ | Очки, VFX, анимации |
| **Прогрессия** | ✅ | Merge → сильнее |
