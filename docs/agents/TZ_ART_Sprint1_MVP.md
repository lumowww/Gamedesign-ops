## 3. Scope of Work

### In-scope Sprint 1

**Башни (2 типа × 3 Merge-уровня = 6 моделей):**
- T-001 Incinerator (Огнемёт) — Lv1, Lv2, Lv3
- T-002 Cryo-Cannon (Крио-Пушка) — Lv1, Lv2, Lv3

**Крипы (2 типа):**
- Корабль-1 (Common) — маленький, быстрый, обтекаемый
- Корабль-2 (Tank) — крупный, широкий, «бронированный»

**Игровое поле:**
- Tile-grid (8×12 слотов): build slot tile (где можно ставить башни)
- Path tile (куда крипы идут — визуально ДРУГОЙ, чем build slot)
- Start point marker (точка спауна крипов)
- End point / база marker (куда нельзя пропустить крипа)

**Particle FX (3 системы):**
- Выстрел башни: Fire (оранжевый) + Cryo (синий) — раздельные
- Попадание в крипа (hit): небольшая вспышка
- Смерть крипа (death): небольшой взрыв / рассыпание

**UI-элементы (только спрайты, верстает UIUX):**
- Иконка башни Огнемёт (для кнопки Spawn)
- Иконка башни Крио (для кнопки Spawn)

**VFX merge:**
- Merge VFX: яркая вспышка + частицы, длительность 0.5–0.8 сек
- Требование: читается с телефона с расстояния 50 см

**Score popup prefab:**
- Число «+N» всплывает вверх и исчезает за 0.8 сек
- Шрифт белый с тёмной обводкой

### Out-of-scope Sprint 1

## 4. Required Output

### Формат
Unity Prefabs + Materials (URP Lit / Simple Lit)
Все меши в одном файле или отдельные .fbx — по удобству ART-агента.

### Стиль — ОБЯЗАТЕЛЬНО ИЗУЧИТЬ
**Art reference board:** https://pin.it/2HReRzicu
**Ключевые требования к стилю:**
- Sci-fi top-down, яркие цвета (не тёмный мрачный) = читаемость на мобиле
- Alien aesthetic: чужеродные формы, нестандартные пропорции
- Cartoon-light: не гиперреализм, лёгкая стилизация
- ❌ НЕ использовать стилистику Random Dice (не кубики, не dice-visual language)
- ❌ НЕ копировать арт любых существующих игр

**Цветовой код башен (ОБЯЗАТЕЛЕН):**
- Incinerator / Огнемёт = красно-оранжевая гамма (#FF4400 — #FF8800)
- Cryo-Cannon / Крио = сине-голубая гамма (#0088FF — #00CCFF)
- Крипы = единый стиль (металлик + чужеродные детали, тёмно-зелёный / фиолетовый)

**Силуэты (принципиально важно):**
- Огнемёт = приземистый, широкий, горизонтальный ствол
- Крио = узкий, вертикальный, с воронкой или раструбом сверху
- Корабль-1 = маленький, обтекаемый, «быстрый» вид
- Корабль-2 = широкий, приплюснутый, «тяжёлый бронированный» вид

### Технические ограничения
```
Polycount башня Lv1:  ≤ 400 tri
Polycount башня Lv2:  ≤ 600 tri
Polycount башня Lv3:  ≤ 800 tri
Polycount крип:       ≤ 300 tri
Polycount tile:       ≤ 50 tri (96 штук на поле!)

Texture atlas:        ВСЁ в одном 1024×1024
                      Башни + крипы + тайлы + маркеры — один атлас
Shader:               URP Simple Lit (без normal maps в Sprint 1)
VFX:                  ≤ 100 particles per system
                      Unity Particle System (НЕ VFX Graph в Sprint 1)
Skinned Mesh:         НЕ нужен для крипов (просто mesh + lerp)
```
