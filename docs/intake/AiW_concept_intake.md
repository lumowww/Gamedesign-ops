## 🎯 Elevator Pitch

**Alien in Waders (AiW)** — это hybrid-casual Tower Defense с merge-механикой, где игрок строит огневые рубежи из инопланетного оружия, сливая башни для создания монстров разрушения.

**Hook:** Хаос под контролем — сливай, разрушай, доминируй.

---

## 🎮 Core Loop

**Micro:** Спаунить башню → ждать волну → сливать две одинаковых → новый тип

**Meta:** Выжить матч → получить шарды → апгрейд колоды → следующий матч сложнее

**Macro:** Открыть 15 башен → PvP рейтинг → Battle Pass → сезонный контент

## 📊 Market Analysis (Quick)

**Жанр:** Tower Defense + Merge

**Конкуренты:**
- Random Dice (Dice Merge + TD) — $10M+ MRR
- Kingdom Rush — классика TD
- Merge Tower — нишевый игрок

**Возможности:**
- Merge механика добавляет глубину классическому TD
- Короткие сессии (3-5 мин) — идеально для mobile
- PvP асинхронный — низкий порог входа

**Риски:**
- Перенасыщенный рынок TD
- CPI может быть высоким ($2-5)
